# CSV SQL-like Query Engine

A lightweight **SQL-like query engine** built in Python for querying CSV files.  
It supports `SELECT`, `WHERE`, `JOIN`, `GROUP BY`, `ORDER BY`, and more using a familiar SQL syntax.
